## ✨ Overview

A simple and interactive **dice game web application** where two players roll dice and the winner is decided instantly.

This project demonstrates **random number generation, DOM manipulation, and dynamic UI updates** using JavaScript.

---

## 🚀 Features

* 🎲 Random dice roll generation
* 👥 Two-player gameplay
* ⚡ Instant result display
* 🎨 Clean and simple UI
* 🔄 Refresh to play again

---

## 🛠️ Tech Stack

| Technology | Purpose    |
| ---------- | ---------- |
| HTML       | Structure  |
| CSS        | Styling    |
| JavaScript | Game logic |

---

## 🎮 How to Play

```bash
1. Open the website
2. Refresh the page
3. Dice rolls automatically
4. Player with higher number wins 🎉
```
